github使用：
    创建仓库   项目描述  创建README.md
    
    创建文件: Create new file
    编辑文件，删除文件
    记录执行
    
    上传文件（一次可以上传多个文件）: Upload files
    
    查找项目中的文件 Find file  (快捷键：T)


    issue
        点击仓库 -> 点击 Issues
        
    回复issue
        先回复，再关闭   可以查看已解决的issues
    
    
    stars ： 收藏此仓库
    watch ： 关注此仓库
    
    pull request： 提交更改到原作者
                   作者审核完可以给个回复（感谢）
